# Harshaa

Premium React Vite homepage for Midas Touch Magick.

## Project Structure

- `src/App.jsx` - main homepage UI
- `src/main.jsx` - React entry point
- `src/styles/styles.css` - global styles and responsive UI
- `src/assets/images/` - hero, logo, service, and transformation images
- `src/assets/icons/` - 3D icon assets
